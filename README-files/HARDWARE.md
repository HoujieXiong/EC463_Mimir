# Mimir: Hardware Components

## Description
For our hardware of Mimir, we have multiple components that is connected our computing source, Raspberry Pi 4.  The components are: <br>
```
    - Raspberry Pi 4
    - LiDAR Camera 
    - Raspberry Pi Camera
    - M/F Jumper Wires 
    - Tactile Buttons
```
